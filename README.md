## Особенности реализации
Язык: C++ (стандарт C++17)
Система сборки: CMake, версия 4.1

## Инструкция по сборке программы
```bash
# Из корневой директории создайте папку build и перейдите в нее
$ mkdir build
$ cd build
# Сконфигурируйте и соберите проект
$ cmake ..
$ cmake --build .
# Запустите программу, указав название тестового файла (testAllForOne.txt | testExample.txt | testQueue.txt)
$ ./task ../testName.txt

# Для проверочных юнит-тестов использовалась библиотека Google Test
# Для запуска тестов указать
 $ ./test_app
